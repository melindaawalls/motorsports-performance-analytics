# motorsports-performance-analytics
Motorsports Telemetry & Track Map Tool

Python project for loading track geometry from CSV files and generating accurate track maps for motorsports analysis. Built to be modular and expandable for future telemetry data integration (speed, throttle, braking, lap timing).

Features
CSV-based track data loading
Dynamic track map visualization
Modular structure for telemetry expansion
Requirements
Python 3.x
numpy
pandas
matplotlib
